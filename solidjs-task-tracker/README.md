# Solid JS Task Tracker Example

This example code is copy of this article - <https://blog.logrocket.com/build-task-tracker-solidjs-typescript/>

## For What?

Just know what is solid js and how is it looks like.

## To Start

To initialize this kind of project, execute the following command.

```sh
npx degit solidjs/templates/ts task-tracker
```

To start this project, execute the following command. Because pnpm is the default package manager of solidjs, it is recommended to use pnpm.

```sh
pnpm install
pnpm dev
```
