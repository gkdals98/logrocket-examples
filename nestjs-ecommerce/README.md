# Nest ecommerce app

This example code is copy of this article - <https://blog.logrocket.com/how-build-ecommerce-app-nestjs/>

## For What?

To learn how consist nest and mongo db infra.

## To Start

Install

```bash
npm install
```

Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
